# Task Manager

Проект React + TypeScript проект под требования из задания.

## Что внутри

- React + TypeScript
- Redux Toolkit + react-redux
- react-router-dom
- axios layer
- Login / Register / Landing / Profile / Dashboard
- Tasks / Task Details / Analytics / Categories / 404
- AuthWrapper и CommonWrapper
- Примеры GET / POST / PUT / PATCH / DELETE
- Глобальный loader и modal ошибок

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Важно

Сейчас проект использует mock API в `src/api/*`, чтобы он запускался сразу без отдельного backend.
